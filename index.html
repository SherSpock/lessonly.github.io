Lesson.ly Coding Style

`.editorconfig`

The app contains an `.editorconfig` file which specifies most of the syntax conventions (spacing, etc.) we use. Be sure you're using [an Editorconfig plugin](http://editorconfig.org/#download) with your preferred text editor to ensure your code conforms to these conventions.
